โหมดเบื้องหลัง (ทดลอง)

ควบคุม CropWat ด้วยการส่งคำสั่งตรง ไม่แตะเมาส์/คีย์บอร์ด/focus ของเครื่องเลย CropWat อยู่หลังหน้าต่างอื่นได้ แต่ห้ามย่อ (minimize) หน้าต่าง CropWat — ถ้ารันแล้วมีปัญหา ให้ติ๊กออกเพื่อกลับไปโหมดปกติ (ดึงหน้าต่างขึ้นมาระหว่างรัน) แล้วบันทึกการตั้งค่า

ความเร็วการรัน

ถ้าเครื่องแฮงค์หรือ CropWat ค้างระหว่างรัน (มักเกิดกับคอมรุ่นเก่า) ให้เปลี่ยนเป็น "ช้า" — เพิ่มเวลาหน่วงระหว่างคำสั่งให้ CropWat ตามทัน แล้วบันทึกการตั้งค่า จากนั้นเริ่มรันใหม่

เลือกไฟล์ Climate/Rain ตามเดือน (Shift-year)

แต่ละวันปลูกที่ทดลองจะใช้ไฟล์ climate/rain ของเดือนนั้นๆ โดยตรง เช่น ปลูกเดือนเมษาใช้ไฟล์ Apr ปลูกเดือนพฤษภาใช้ไฟล์ May — ถ้าไม่มีไฟล์เดือนนั้นจริงๆ จะถอยไปใช้ไฟล์เดือนก่อนหน้าที่ใกล้ที่สุดให้เองอัตโนมัติ (เช่น ไม่มีไฟล์ Sep จะใช้ Aug แทน) — ปิดสวิตช์นี้เพื่อกลับไปพฤติกรรมเดิม (ทั้งปีใช้ไฟล์เดือนแรกสุดที่ทดลองปลูกไฟล์เดียวตลอดทั้งปี)

⏱ เวลาทำมือเทียบเท่า

ใช้คำนวณตัวเลข "ชม. ทำมือเทียบเท่า" ใน Dashboard — นาทีต่อวันปลูก 1 วัน (รวมทุกขั้นตอน: หาไฟล์ รัน print ถ่ายภาพ กรอก Excel) × จำนวนวันปลูกทั้งหมด

Override การเลือกสถานี/ไฟล์ (ถ้าจำเป็น)
เว้นว่างไว้ถ้าสแกนแล้วเจอตัวเลือกเดียว ระบบจะเลือกให้เองอัตโนมัติ — กรอกตรงนี้เฉพาะตอนเจอมากกว่า 1 ตัวเลือกแล้วต้องระบุเอง

เริ่มใช้งานยังไง

  1. เตรียมข้อมูลไว้ในโฟลเดอร์เดียว — ชื่อไฟล์ต้องตามกติกานี้
    ระบบอ่าน "ปี+เดือน" จากชื่อไฟล์โดยตรง ชื่อไม่ตรงกติกา = สแกนไม่เจอ (ตัวใหญ่-เล็กไม่สน จะซ้อนโฟลเดอร์ย่อยกี่ชั้นก็ได้):
    • โฟลเดอร์สถานี: ขึ้นต้น Clim_ และ Rain_ เช่น Clim_425201
    • ไฟล์ climate: {รหัสสถานี}clim_{ปี}{เดือน3ตัว}.PED เช่น 425201clim_1981apr.PED
    • ไฟล์ rain: rain_{รหัสสถานี}_{ปี}{เดือน3ตัว}.CRD เช่น Rain_450006_1981Apr.CRD
    • ไฟล์พืช/ดิน: ชื่ออะไรก็ได้ ขอแค่นามสกุล .CRO / .SOI
  2. ตั้งค่าครั้งแรก (แท็บ "ตั้งค่า")
    วาง path โฟลเดอร์ข้อมูล → กด สแกน ให้ระบบตรวจว่าเจอครบ → วาง path โฟลเดอร์ผลลัพธ์ → กด บันทึกการตั้งค่า (ทำครั้งเดียว ค่าอยู่ถาวร)
  3. เปิดโปรแกรม CropWat 8.0 ทิ้งไว้
    แค่เปิดโปรแกรมเปล่าๆ พอ ไม่ต้องเปิดไฟล์อะไรเลย — ระบบเปิดไฟล์ให้เองทั้งหมด
  4. เริ่มรัน (แท็บ "Dashboard")
    เลือกช่วงปี + วันปลูกในปฏิทิน → กด เริ่มรันทั้งหมด (หรือกด Ctrl+Alt+F9 จากที่ไหนก็ได้) — โหมดเบื้องหลัง (ค่าเริ่มต้น) ใช้คอมทำงานอื่นไปได้เลยระหว่างรัน แค่อย่าย่อ (minimize) หน้าต่าง CropWat วางไว้หลังหน้าต่างอื่นได้ ดูความคืบหน้าจากแถบลอยมุมขวาล่างจอ (Ctrl+Alt+F10 = หยุด)
  5. เสร็จแล้วสร้างไฟล์ผลลัพธ์
    กด สร้าง/อัปเดต Excel (ตาราง Result) และ สร้างไฟล์ Word (ภาพ) (รวม screenshot) — กดซ้ำได้ทุกเมื่อ สร้างใหม่จากข้อมูลที่มีอยู่

มีเวอร์ชันใหม่

แนะนำให้อัปเดตทันทีเพื่อได้ตัวแก้บั๊กล่าสุด — โปรแกรมจะปิดและเปิดใหม่เองอัตโนมัติ
⚠ ถ้าอัปเดตแล้วเกิดข้อผิดพลาดหรือโปรแกรมไม่เปิดขึ้นมาเอง: ปิดโปรแกรม (ปุ่ม ✕ บนแถบลอย หรือดับเบิลคลิกไอคอนโปรแกรมอีกครั้ง) แล้วเปิดใหม่ จากนั้นลองกดอัปเดตอีกครั้ง

กำลังดาวน์โหลดเวอร์ชันใหม่...

อย่าปิดโปรแกรมระหว่างนี้ — เมื่อดาวน์โหลดเสร็จ โปรแกรมจะปิดและเปิดใหม่เอง หน้านี้จะกลับมาโดยอัตโนมัติ
