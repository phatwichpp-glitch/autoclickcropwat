# คู่มือรันใน Virtual Machine (ทางเลือกสำหรับรันเบื้องหลังแบบชัวร์ 100%)

ตั้งแต่ v0.5.0 โปรแกรมมี **โหมดเบื้องหลัง** ในตัวแล้ว (ค่าเริ่มต้น) — ใช้คอมทำงานอื่น
ระหว่างรันได้โดยไม่ต้องทำอะไรเพิ่ม **ลองโหมดนั้นก่อน** คู่มือนี้เป็นแผนสำรองสำหรับ
กรณีที่โหมดเบื้องหลังใช้กับเครื่อง/เวอร์ชัน CropWat ของคุณไม่ได้ หรืออยากแยกงาน
ออกจากเครื่องหลักแบบเด็ดขาด

หลักการ: รันทั้ง CropWat + CropWatAutoRunner ไว้ใน "คอมจำลอง" (VM) ซึ่งมีจอ/เมาส์/
คีย์บอร์ดของตัวเองแยกขาดจากเครื่องจริง — ในนั้นโปรแกรมจะคลิกจะแย่งหน้าจอยังไงก็ไม่
กระทบเครื่องจริงเลย แม้แต่ย่อหน้าต่าง VM ทิ้งไว้งานก็เดินต่อ

## ขั้นตอน (ทำครั้งเดียว ~1 ชั่วโมง)

1. **ติดตั้ง VirtualBox** (ฟรี ใช้ได้บน Windows Home): https://www.virtualbox.org/
2. **สร้าง VM Windows**
   - ดาวน์โหลด Windows 10/11 ISO จากเว็บ Microsoft (Media Creation Tool / ดาวน์โหลด ISO ตรง)
   - ใน VirtualBox: New → เลือก ISO → ตั้ง Memory 4 GB, CPU 2 ตัว, ดิสก์ 60 GB
   - ติดตั้ง Windows ตามปกติ (ต้องมี license Windows สำหรับ VM ตามเงื่อนไข Microsoft)
3. **ติดตั้ง Guest Additions** (เมนู Devices → Insert Guest Additions CD) — ทำให้ปรับ
   ขนาดจอ/แชร์โฟลเดอร์/คลิปบอร์ดได้
4. **แชร์โฟลเดอร์ข้อมูล**: Settings ของ VM → Shared Folders → เพิ่มโฟลเดอร์ข้อมูล
   CropWat ของเครื่องจริง (ติ๊ก Auto-mount) — ใน VM จะเห็นเป็นไดรฟ์ เช่น `Z:\`
5. **ใน VM**: ติดตั้ง CropWat 8.0 + วาง `CropWatAutoRunner.exe` → เปิดโปรแกรม →
   ตั้งค่าโฟลเดอร์ต้นทางชี้ไปที่โฟลเดอร์แชร์ → ใช้งานเหมือนบนเครื่องจริงทุกอย่าง
6. **ระหว่างรัน**: ย่อ/วางหน้าต่าง VM ไว้มุมจอได้เลย เครื่องจริงใช้งานอิสระ 100%
   (แนะนำตั้งโฟลเดอร์ผลลัพธ์ไว้ในโฟลเดอร์แชร์ด้วย จะได้เปิดไฟล์ผลจากเครื่องจริงได้ทันที)

## ข้อควรรู้

- CropWat เบามาก (โปรแกรมยุค 2009) — VM สเปกต่ำสุดก็รันสบาย
- ถ้าเครื่องจริง RAM 8 GB ขึ้นไปถึงจะแนะนำ (VM กิน 4 GB)
- การอัปเดต CropWatAutoRunner ใน VM ก็กดปุ่มอัปเดตในโปรแกรมได้ปกติ (ต้องต่อเน็ต)
