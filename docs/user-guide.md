## 1. สิ่งที่ต้องมีก่อนเริ่ม

| รายการ | รายละเอียด |
|---|---|
| Windows 10 หรือ 11 | โปรแกรมทำงานบน Windows เท่านั้น (ควบคุม CropWat ผ่านกลไกของ Windows โดยตรง) |
| CropWat 8.0 | โปรแกรมของ FAO ตัวจริง ติดตั้งไว้ในเครื่องก่อน — ดาวน์โหลดได้จาก [เว็บไซต์ FAO](https://www.fao.org/land-water/databases-and-software/cropwat/en/) (CropWat Auto-runner ทำหน้าที่ "ควบคุม" CropWat ให้ ไม่ได้แทนที่มัน) |
| ไฟล์ข้อมูล | climate / rain / crop / soil ที่จะใช้ประมวลผล (กติกาตั้งชื่อไฟล์อยู่ข้อ 4) |

> **คำแนะนำ:** ตอนติดตั้ง CropWat 8.0 ให้กด Next ตามค่าเริ่มต้นของตัวติดตั้งไป
> เรื่อยๆ (อย่าเปลี่ยนโฟลเดอร์ติดตั้ง) แล้วข้อ 3 ด้านล่างจะ "ไม่ต้องทำอะไรเลย"

## 2. ติดตั้งและเปิดโปรแกรมครั้งแรก

1. ดาวน์โหลด `CropWatAutoRunner.exe` เวอร์ชันล่าสุดจากหน้า [Releases](../../releases)
   — เป็นไฟล์เดียวจบ **ไม่ต้องติดตั้ง** วางไว้ที่ไหนก็ได้ (แนะนำสร้างโฟลเดอร์
   ของมันเอง เพราะโปรแกรมจะเขียนไฟล์ log ไว้ข้างๆ ตัว .exe)
2. ดับเบิลคลิกเปิด — ถ้า Windows ขึ้นเตือน **"Windows protected your PC"**
   (SmartScreen) ให้กด **More info** แล้วกด **Run anyway** (ขึ้นเพราะไฟล์ไม่ได้
   ซื้อใบรับรองลงนามโค้ด ไม่ได้แปลว่ามีไวรัส)
3. หน้าต่างโปรแกรมจะเปิดขึ้นมาเอง พร้อมไอคอนโปรแกรมที่**ถาดระบบ (System Tray)**
   มุมล่างขวาของจอ — ไอคอนนี้คือสัญญาณว่าโปรแกรมทำงานอยู่ ปิดหน้าต่างไปโปรแกรม
   ก็ยังทำงานต่อเบื้องหลัง (ปิดโปรแกรมจริงๆ ให้คลิกขวาที่ไอคอน → "ปิดโปรแกรม")
4. เปิดครั้งแรกจะมี**คู่มือเริ่มต้นใช้งาน**เด้งขึ้นมาสรุปขั้นตอนทั้งหมด — เรียกดู
   ซ้ำได้ตลอดจากปุ่ม **"คู่มือ"** มุมขวาบน

## 3. ต้องตั้งตำแหน่ง (path) โปรแกรม CropWat เองไหม?

**คำตอบสั้น: ส่วนใหญ่ไม่ต้อง** — ระบบค้นหาให้อัตโนมัติ

| สถานการณ์ | ต้องทำอะไร |
|---|---|
| ติดตั้ง CropWat 8.0 แบบมาตรฐาน (กด Next ตามตัวติดตั้ง) | **ไม่ต้องทำอะไรเลย** เว้นช่องว่างไว้ได้ ระบบหาเจอเองที่ `C:\Program Files (x86)\CROPWAT\cropwat.exe` |
| ติดตั้งไว้โฟลเดอร์อื่น / ไดรฟ์อื่น | ไปที่แท็บ **ตั้งค่า** → การ์ด **"โปรแกรม CropWat 8.0"** → กด **เลือกไฟล์** แล้วชี้ไปที่ `CROPWAT.exe` ของคุณ → **บันทึกการตั้งค่า** |
| ไม่แน่ใจว่าติดตั้งแบบไหน | ดูที่การ์ด "โปรแกรม CropWat 8.0" ในแท็บตั้งค่า — มีบรรทัดสถานะบอกชัดเจน (ดูด้านล่าง) |

**วิธีเช็คว่าระบบหาเจอหรือยัง:** เปิดแท็บ **ตั้งค่า** → ดูใต้ช่องกรอกในการ์ด
"โปรแกรม CropWat 8.0":

- ✓ สีเขียว **"พบโปรแกรมอัตโนมัติที่ C:\Program Files (x86)\CROPWAT\cropwat.exe
  — ไม่ต้องตั้งค่าใดๆ เพิ่ม"** → พร้อมใช้งาน จบ ไม่ต้องแตะอะไร
- ✓ สีเขียว **"ใช้ไฟล์ที่ระบุไว้: ..."** → ใช้ path ที่คุณตั้งเอง พร้อมใช้งาน
- ✗ สีแดง **"ยังไม่พบโปรแกรม CropWat 8.0 ในเครื่อง"** → ยังใช้งานไม่ได้
  ต้องติดตั้ง CropWat ก่อน หรือถ้าติดตั้งแล้วให้กด "เลือกไฟล์" ชี้ตำแหน่งเอง

**หาไฟล์ CROPWAT.exe ของตัวเองไม่เจอ?** คลิกขวาที่ไอคอน CropWat 8.0 บน
Desktop/Start Menu → **Open file location** จะพาไปที่โฟลเดอร์ที่มีไฟล์ .exe อยู่

## 4. เตรียมไฟล์ข้อมูล — กติกาตั้งชื่อ (สำคัญมาก)

ระบบอ่าน "ปี + เดือน" จาก**ชื่อไฟล์โดยตรง** ชื่อไม่ตรงกติกา = ตรวจสอบไม่พบ
(ตัวพิมพ์ใหญ่/เล็กไม่มีผล จะซ้อนโฟลเดอร์ย่อยกี่ชั้นก็ได้):

| ประเภท | กติกา | ตัวอย่างที่ถูก |
|---|---|---|
| โฟลเดอร์สถานี | ขึ้นต้น `Clim_` หรือ `Rain_` | `Clim_425201`, `Rain_450006` |
| ไฟล์ climate | `{รหัสสถานี}clim_{ปี}{เดือน3ตัว}.PED` | `425201clim_1981apr.PED` |
| ไฟล์ rain | `rain_{รหัสสถานี}_{ปี}{เดือน3ตัว}.CRD` | `Rain_450006_1981Apr.CRD` |
| ไฟล์พืช (crop) | นามสกุล `.CRO` ชื่ออะไรก็ได้ | `MAIZE.CRO` |
| ไฟล์ดิน (soil) | นามสกุล `.SOI` ชื่ออะไรก็ได้ | `SANDYLOAM.SOI` |

- เดือน 3 ตัว = `jan feb mar apr may jun jul aug sep oct nov dec`
- รวมทุกอย่าง (โฟลเดอร์สถานี + ไฟล์พืช + ไฟล์ดิน) ไว้ใน**โฟลเดอร์ใหญ่โฟลเดอร์เดียว**
  แล้วชี้โปรแกรมไปที่โฟลเดอร์นั้นโฟลเดอร์เดียวพอ — ระบบสแกนลึกลงไปเอง

**ตัวอย่างโครงสร้างที่ใช้ได้:**

```
ข้อมูลของฉัน/                      ← ชี้โปรแกรมมาที่นี่
  Clim_425201/
    425201clim_1981apr.PED
    425201clim_1981may.PED
    ...
  Rain_450006/
    Rain_450006_1981Apr.CRD
    ...
  MAIZE.CRO
  SANDYLOAM.SOI
```

## 5. ตั้งค่าครั้งแรก (แท็บ "ตั้งค่า")

ทำตามลำดับ 5 ขั้น — ทำครั้งเดียว ระบบจดจำถาวรแม้ปิดโปรแกรม:

1. **โฟลเดอร์ข้อมูลต้นทาง:** กด **เลือกโฟลเดอร์** ชี้ไปที่โฟลเดอร์ใหญ่จากข้อ 4
   แล้วกด **ตรวจสอบ** — ต้องเห็นรายการสถานี/ไฟล์ครบตามที่คาด ถ้าไม่ครบให้กลับไป
   เช็คชื่อไฟล์ตามกติกาข้อ 4
2. **โปรแกรม CropWat 8.0:** ดูบรรทัดสถานะ — ถ้าขึ้น ✓ สีเขียวอยู่แล้ว ข้ามไปได้เลย
   (ส่วนใหญ่จะเป็นแบบนี้ ดูข้อ 3)
3. **โฟลเดอร์ผลลัพธ์:** กด **เลือกโฟลเดอร์** ชี้ไปที่โฟลเดอร์เปล่าที่จะเก็บผลลัพธ์
   ทั้งหมด (ระบบสร้างโครงสร้างข้างในให้เอง)
4. ตัวเลือกอื่นๆ **แนะนำให้คงค่าเริ่มต้นไว้ทั้งหมด** (โหมดเดสก์ท็อปเบื้องหลัง = เปิด,
   สร้างไฟล์ผลลัพธ์อัตโนมัติ = เปิด, ความเร็ว = ปกติ, Shift-year = เปิด)
5. กด **บันทึกการตั้งค่า**

> **เครื่องรุ่นเก่า/ช้า:** ถ้าเจอ CropWat ไม่ตอบสนองระหว่างประมวลผลบ่อยๆ ให้เปลี่ยน
> "ความเร็วการประมวลผล" เป็น **ช้า** แล้วบันทึก จากนั้นเริ่มประมวลผลใหม่

## 6. เลือกช่วงปีและวันปลูก (ปฏิทิน)

ที่แท็บ **ภาพรวม** การ์ด "กำหนดการประมวลผล":

1. ใส่**ปีเริ่มต้น–ปีสิ้นสุด** ที่ต้องการทดสอบ (เช่น 1981 → 2025)
2. ในปฏิทินแต่ละเดือน เลือกวันปลูกที่ต้องการทดสอบ — มีปุ่มลัด: **ทุกวัน / ทุก 7 วัน /
   วันที่ 1, 15 / กำหนดเอง** (กำหนดเอง = คลิกเลือกรายวันเองในตาราง)
3. เดือนไหนไม่ต้องการทดสอบ ติ๊ก **ข้าม** ที่มุมขวาบนของเดือนนั้น
4. แถว **Capture** ของแต่ละเดือน = เลือกว่าวันไหนให้บันทึกภาพหน้าจอผลลัพธ์
   (Screenshot) เก็บไว้ประกอบรายงานด้วย — จุดสีทองบนตัวเลขวัน = วันที่จะบันทึกภาพ
5. ปฏิทิน**บันทึกอัตโนมัติ**ทุกครั้งที่แก้ ไม่ต้องกดปุ่มบันทึก
6. แถบสรุปด้านล่างบอกจำนวนวันปลูก/ปี, รอบคำนวณทั้งหมด, จำนวนภาพบันทึก และ
   ชั่วโมงงานที่ประหยัดได้เทียบกับการทำมือ

## 7. เริ่มประมวลผลและติดตามงาน

1. กด **เริ่มประมวลผล** (หรือกด `Ctrl+Alt+F9` จากที่ใดก็ได้ในเครื่อง)
2. ก่อนเริ่มจริง ระบบ**ตรวจไฟล์ climate/rain ล่วงหน้าครบทุกวันปลูก** — ถ้าพบวันปลูก
   ที่ไม่มีไฟล์ให้ใช้ (เช่น สถานีเริ่มมีข้อมูลทีหลังเดือนที่เลือก) จะขึ้นกล่องแจ้ง
   รายละเอียดให้ยืนยันก่อน (วันปลูกที่มีปัญหาจะขึ้นสถานะ "มีปัญหา" ส่วนวันอื่น
   ดำเนินการตามปกติ)
3. ระบบจะ**เปิด CropWat ให้เองในเดสก์ท็อปแยกเบื้องหลัง** ที่มองไม่เห็นบนจอ —
   **ใช้เครื่องทำงานอื่นได้ตามปกติ** ระหว่างประมวลผล การคลิก/พิมพ์ใดๆ ไม่กระทบงาน

**ติดตามความคืบหน้าได้ 4 ทาง:**

| ช่องทาง | รายละเอียด |
|---|---|
| การ์ด "ภาพหน้าจอ CropWat" | แสดงภาพการทำงานจริงโดยอัตโนมัติทุกครั้งที่ระบบบันทึกภาพ — กด **Snapshot** ดูภาพ ณ ตอนนี้ทันที หรือกด **Live View** ดูต่อเนื่อง (อัปเดตทุก ~2 วินาที; งานช้าลงเล็กน้อยระหว่างเปิด และหยุดเองเมื่องานเสร็จ) คลิกที่ภาพเพื่อขยาย |
| แถบความคืบหน้าลอย | มุมล่างขวาของจอ โผล่เองระหว่างประมวลผล ลากย้ายได้ กด 🗕 เพื่อซ่อน (เรียกคืน: คลิกขวาไอคอนถาดระบบ → "แสดงแถบความคืบหน้า") |
| ไอคอนถาดระบบ (System Tray) | เอาเมาส์ชี้ = เห็นปี/เปอร์เซ็นต์/เวลาที่เหลือ โดยไม่ต้องเปิดหน้าต่างใด — จุดเขียวบนไอคอน = กำลังทำงาน |
| หน้าต่างโปรแกรม | แถบความคืบหน้า + สถานะรายปี — กด **ย่อหน้าต่าง** มุมขวาบนเพื่อพับเก็บ โปรแกรมทำงานต่อเบื้องหลัง (เปิดกลับ: ดับเบิลคลิกไอคอนถาดระบบ) |

- หยุดงาน: ปุ่ม **หยุด** หรือ `Ctrl+Alt+F10` — งานที่เสร็จแล้วไม่หาย
- **งานเสร็จมี notification เด้งแจ้งทันที** แม้กำลังทำงานอื่นอยู่
- **ทำต่อจากเดิมได้เสมอ:** กด "เริ่มประมวลผล" อีกครั้งเมื่อไรก็ได้ ระบบข้ามรายการ
  ที่เสร็จสมบูรณ์แล้วอัตโนมัติ (ดูได้จากกล่อง "รายการที่เสร็จสมบูรณ์" ในหน้าภาพรวม)

## 8. ไฟล์ผลลัพธ์

เมื่อประมวลผลเสร็จครบ ระบบสร้างไฟล์ส่งงานให้**อัตโนมัติ** (ปิดได้ที่แท็บตั้งค่า):

```
โฟลเดอร์ผลลัพธ์/
  txt/               ← ไฟล์ผลดิบจาก CropWat (รายวันปลูก)
  screenshots/       ← ภาพบันทึกหน้าจอ (เฉพาะวันที่เลือก Capture ไว้)
  Result.xlsx        ← ตารางรวมผลทุกวันปลูก
  Screenshots.docx   ← เอกสารรวมภาพทั้งหมด
```

- เปิดโฟลเดอร์เร็วๆ: ปุ่ม **เปิดโฟลเดอร์** ในหน้าภาพรวม หรือคลิกขวาไอคอนถาดระบบ
  → "เปิดโฟลเดอร์ผลลัพธ์"
- สร้างซ้ำเมื่อไรก็ได้: ปุ่ม **สร้าง Excel** / **สร้าง Word** — สร้างใหม่จากไฟล์
  ที่มีอยู่จริงเสมอ ไม่ต้องรันใหม่
- โฟลเดอร์ `_peek_screenshots` (ถ้ามี) = ภาพจากปุ่ม Snapshot/Live View ใช้ตรวจสอบ
  เท่านั้น **ไม่เกี่ยวกับไฟล์ส่งงาน** ลบทิ้งได้เสมอ

## 9. ปัญหาที่พบบ่อย

**เริ่มประมวลผลแล้วขึ้นว่า "หาไฟล์โปรแกรม CropWat ไม่เจอ"**
ยังไม่ได้ติดตั้ง CropWat 8.0 หรือติดตั้งไว้นอกตำแหน่งมาตรฐาน — ดูข้อ 3
(ตั้ง path ที่การ์ด "โปรแกรม CropWat 8.0" ในแท็บตั้งค่า)

**กด "เริ่มประมวลผล" แล้วขึ้นกล่องเตือนว่าบางวันปลูก "ไม่มีไฟล์"**
สถานี climate/rain มีข้อมูลเริ่มทีหลังเดือนที่เลือกทดสอบ — ระบบจะไม่เดาใช้ข้อมูล
เดือนอื่นแทนเด็ดขาด (ผลจะเพี้ยน) เลือกเองว่าจะข้ามวันปลูกเหล่านั้นหรือเปลี่ยน
แผนการทดสอบ

**ตรวจสอบโฟลเดอร์ข้อมูลแล้วไม่เจอไฟล์ / เจอไม่ครบ**
เกือบทั้งหมดเกิดจากชื่อไฟล์ไม่ตรงกติกา — เทียบกับตารางในข้อ 4 ทีละส่วน
(โฟลเดอร์สถานีต้องขึ้นต้น `Clim_`/`Rain_` ด้วย)

**CropWat ไม่ตอบสนอง / ค้างระหว่างประมวลผล**
กดปุ่ม **Force Close** ในหน้าภาพรวม — บังคับปิด CropWat ทุกตัว + คืนสถานะระบบ
พร้อมเริ่มใหม่ทันที ข้อมูลที่เสร็จแล้วไม่หาย กด "เริ่มประมวลผล" ต่อจากเดิมได้เลย
(CropWat crash เองเป็นครั้งคราวจากบั๊กภายในของมัน เช่น "Access violation" —
ระบบเปิด CropWat ใหม่ให้เองทุกครั้งที่เริ่มงาน จึงฟื้นจากเหตุการณ์แบบนี้ได้ง่าย)
ถ้าเครื่องช้าและเจอบ่อย ให้ปรับ "ความเร็วการประมวลผล" เป็น **ช้า** ในแท็บตั้งค่า

**ขึ้น error "เจอหน้าต่าง...มากกว่า 1 หน้าต่าง"**
พบในเวอร์ชันก่อน v0.10.1 เมื่อโปรแกรมถูกปิดกลางคันระหว่างงาน (CropWat เก่าค้าง
อยู่เบื้องหลัง) — **อัปเดตเป็นเวอร์ชันล่าสุด** ระบบจะเก็บกวาดตัวค้างให้เอง
อัตโนมัติทุกครั้งก่อนเริ่ม (เวอร์ชันเก่า: กด Force Close หนึ่งครั้งแล้วเริ่มใหม่)

**บางปี/บางวันปลูกขึ้นสถานะ "มีปัญหา" — อยากรู้ว่า CropWat แจ้ง error อะไร**
ตั้งแต่ v0.11.0 เมื่อ CropWat เด้ง error ระหว่างประมวลผลเบื้องหลัง ระบบจะ**ถ่ายภาพ
เก็บให้อัตโนมัติ** — ตั้งแต่ v0.11.2 เป็นภาพ**หน้าต่าง CropWat ทั้งบาน**พร้อมกล่อง
error วางทับตรงกลาง (เห็นทั้งข้อความ error และข้อมูลที่โหลดอยู่ตอนนั้นในภาพเดียว) —
ในรายการปีที่มีปัญหา กดปุ่ม **📷 ดูภาพ Error** เพื่อเปิดดูได้ว่า CropWat บ่นอะไร
และพังตอนโหลดข้อมูลอะไร (ภาพเก็บที่โฟลเดอร์ผลลัพธ์ → `_error_dialogs`)
error ประเภทนี้ส่วนใหญ่เป็น**ปัญหาที่ตัวข้อมูลต้นทาง**ที่ CropWat เองไม่ยอมรับ เช่น
"Rain data (January) are not OK" = ไฟล์ rain ของสถานะนั้นมีข้อมูลไม่ครบ/ผิดรูปแบบ
ในเดือนที่ระบุ ต้องแก้ที่ไฟล์ข้อมูล ไม่ใช่ที่โปรแกรม (โฟลเดอร์ `_error_dialogs`
ลบทิ้งได้เสมอ ไม่เกี่ยวกับไฟล์ส่งงาน)

**เปิดโปรแกรมแล้วค้างที่ "กำลังเชื่อมต่อ..."**
ปิดหน้าต่างโปรแกรม แล้วดับเบิลคลิกไอคอนโปรแกรมเปิดใหม่อีกครั้ง — ถ้ายังไม่หาย
ให้ปิดโปรแกรมจริงๆ (คลิกขวาไอคอนถาดระบบ → ปิดโปรแกรม) แล้วเปิดใหม่

**อยากแยกขาดจากเครื่องจริง 100% (ทางเลือกขั้นสุด)**
ดู `docs/vm-guide.md` — วิธีรันใน Virtual Machine (หนักกว่าโหมดปกติมาก ใช้เฉพาะ
กรณีโหมดเดสก์ท็อปเบื้องหลังใช้กับเครื่อง/เวอร์ชัน CropWat ของคุณไม่ได้จริงๆ)

## 10. การอัปเดตโปรแกรม

- โปรแกรมตรวจเวอร์ชันใหม่ให้อัตโนมัติ (ต้องต่ออินเทอร์เน็ต) มีใหม่จะเด้งแจ้งใน
  โปรแกรม กด **อัปเดตทันที** ได้เลย — ดาวน์โหลดเสร็จโปรแกรมปิด-เปิดใหม่เอง
- กดตรวจเองได้ที่ปุ่ม **ตรวจอัปเดต** มุมขวาบน
- ⚠ **อย่าอัปเดตระหว่างกำลังประมวลผล** — งานรอบนั้นจะหยุดกลางคัน (ข้อมูลที่เสร็จ
  แล้วไม่หาย เริ่มต่อจากเดิมได้ แต่เสียเวลารอบใหม่) รอให้งานเสร็จก่อนค่อยอัปเดต
