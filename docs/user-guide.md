# คู่มือเริ่มต้นใช้งาน CropWat Auto-runner

คู่มือนี้เขียนสำหรับผู้ใช้ทั่วไป (ไม่ต้องเป็นโปรแกรมเมอร์) — ถ้าอยากรู้รายละเอียด
ทางเทคนิค/โครงสร้างโปรแกรม ดูที่ `README.md` แทน

## สิ่งที่ต้องมีก่อนเริ่ม

- Windows (โปรแกรมนี้ทำงานบน Windows เท่านั้น เพราะควบคุม CropWat ผ่านกลไกของ
  Windows โดยตรง)
- ติดตั้ง [CropWat 8.0](https://www.fao.org/land-water/databases-and-software/cropwat/en/)
  ไว้แล้ว (โปรแกรมของ FAO ตัวจริง — CropWat Auto-runner แค่ควบคุมมันให้ ไม่ได้
  แทนที่มัน)
- ข้อมูล climate/rain/crop/soil ที่จะใช้รัน (ดูกติกาตั้งชื่อไฟล์ด้านล่าง)

## 1. ดาวน์โหลดและเปิดโปรแกรม

1. ไปที่หน้า [Releases](../../releases) แล้วดาวน์โหลด `CropWatAutoRunner.exe`
   เวอร์ชันล่าสุด (ไฟล์เดียว ไม่ต้องติดตั้ง)
2. เปิดไฟล์ — ถ้า Windows ขึ้นเตือน **"Windows protected your PC"**
   (SmartScreen) ให้กด **More info** แล้วกด **Run anyway** — ขึ้นแบบนี้เพราะไฟล์
   ไม่ได้ซื้อใบรับรองลงนามโค้ด (code signing) ไม่ได้แปลว่ามีไวรัส
3. โปรแกรมจะเปิดหน้าต่างเบราว์เซอร์ขึ้นมาเอง (เป็นหน้าจอของโปรแกรม ทำงานบน
   เครื่องเราเอง ไม่ได้ส่งข้อมูลออกไปไหน)

## 2. เตรียมข้อมูล — กติกาตั้งชื่อไฟล์ (สำคัญมาก)

ระบบอ่าน "ปี + เดือน" จากชื่อไฟล์โดยตรง **ชื่อไม่ตรงกติกา = สแกนไม่เจอ** (ตัวใหญ่/
เล็กไม่มีผล จะซ้อนโฟลเดอร์ย่อยกี่ชั้นก็ได้ ไม่ต้องเรียงเหมือนกันทุกสถานี):

| ประเภท | กติกา | ตัวอย่าง |
|---|---|---|
| โฟลเดอร์สถานี | ขึ้นต้น `Clim_` หรือ `Rain_` | `Clim_425201`, `Rain_450006` |
| ไฟล์ climate | `{รหัสสถานี}clim_{ปี}{เดือน3ตัว}.PED` | `425201clim_1981apr.PED` |
| ไฟล์ rain | `rain_{รหัสสถานี}_{ปี}{เดือน3ตัว}.CRD` | `Rain_450006_1981Apr.CRD` |
| ไฟล์พืช/ดิน | นามสกุล `.CRO` / `.SOI` ชื่ออะไรก็ได้ | `MAIZE.CRO`, `SANDYLOAM.SOI` |

รวมทุกอย่าง (สถานี + พืช + ดิน) ไว้ใน **โฟลเดอร์เดียว** — โปรแกรมสแกนลึกเข้าไปเอง

## 3. ตั้งค่าครั้งแรก (แท็บ "ตั้งค่า")

1. วาง path โฟลเดอร์ข้อมูลต้นทาง (ข้อ 2) ที่ "โฟลเดอร์ข้อมูลต้นทาง" แล้วกด
   **สแกน** — ตรวจว่าเจอสถานี/ไฟล์ครบตามที่คาด
2. วาง path โฟลเดอร์ที่จะเก็บผลลัพธ์ที่ "โฟลเดอร์ผลลัพธ์"
3. กด **บันทึกการตั้งค่า** — ทำครั้งเดียว ค่าจะอยู่ถาวรแม้ปิดโปรแกรมไปแล้วเปิดใหม่

## 4. ไม่ต้องเปิด CropWat เอง (โหมดเดสก์ท็อปซ่อน — ค่าเริ่มต้นตั้งแต่ v0.7.0)

โปรแกรมจะ**เปิด CropWat ให้เองในเดสก์ท็อปซ่อน**ที่แยกจากจอของคุณ 100% — คุณ
ใช้คอมทำงานอื่นได้เต็มที่ระหว่างรัน เผลอคลิก/พิมพ์อะไรก็ไม่กระทบการรันเลย
(แค่ตั้ง path ไฟล์ `.exe` ของ CropWat ที่การ์ด "โปรแกรม CropWat 8.0" ครั้งแรก —
ถ้าติดตั้ง CropWat ไว้ที่ตำแหน่งมาตรฐาน ระบบหาให้เองอัตโนมัติ)

ต้องการตรวจสอบว่า CropWat กำลังทำอะไรอยู่ ดูได้จากการ์ด **"ภาพหน้าจอ CropWat"**
ในหน้าภาพรวม — ระบบแสดงภาพโดยอัตโนมัติทุกครั้งที่บันทึกภาพตามปฏิทิน กดปุ่ม
**"ถ่ายภาพสด"** เพื่อดูสถานะปัจจุบันทันที หรือกดปุ่ม **"ดูสด"** เพื่อดูการทำงาน
แบบต่อเนื่อง (ภาพอัปเดตใหม่ทุก ~2 วินาที — การประมวลผลอาจช้าลงเล็กน้อยระหว่าง
เปิดดูสด ระบบหยุดดูสดให้เองเมื่อประมวลผลเสร็จ) ภาพตรวจสอบเหล่านี้เก็บแยกจาก
ภาพบันทึกที่ใช้ส่งงานคนละไฟล์กัน คลิกที่ภาพเพื่อขยายดูเต็มจอได้

ถ้าปิดโหมดเดสก์ท็อปซ่อน (แท็บตั้งค่า) = โหมดคลาสสิก: ต้องเปิด CropWat 8.0
ค้างไว้เองก่อนเริ่มรัน และหน้าต่าง CropWat จะถูกดึงขึ้นมาระหว่างรัน (ไม่ควรใช้
เครื่องทำอย่างอื่นระหว่างนั้น)

## 5. เลือกวันปลูกที่จะทดลอง + เริ่มรัน (แท็บ "Dashboard")

1. เลือกช่วงปีและวันปลูกที่ต้องการทดลองในปฏิทิน (ปฏิทินบันทึกอัตโนมัติ)
2. กด **เริ่มประมวลผล** (หรือกด `Ctrl+Alt+F9` จากที่ไหนก็ได้ในเครื่อง)
3. ก่อนเริ่มจริง ระบบจะ**สแกนตรวจไฟล์ climate/rain ให้ครบทุกวันปลูกที่วางแผนไว้
   ก่อนเสมอ** — ถ้าเจอวันปลูกที่ไม่มีไฟล์ข้อมูลย้อนหลังให้ใช้ (เช่น สถานีเริ่มมี
   ข้อมูลทีหลังเดือนที่เลือก) จะขึ้นกล่องแจ้งเตือนรายละเอียดให้ก่อน ต้องกด
   ยืนยันเองถึงจะเริ่มรันได้ (วันปลูกที่มีปัญหาจะขึ้นสถานะ "มีปัญหา" ส่วนวันปลูก
   อื่นที่มีไฟล์ครบจะรันตามปกติ)
4. ระหว่างรันใช้คอมทำงานอื่นได้เต็มที่ (โหมดเดสก์ท็อปซ่อน — CropWat ไม่อยู่บน
   จอของคุณเลย)
5. ดูความคืบหน้าได้จาก: แถบลอยมุมขวาล่างจอ, ไอคอนถาดระบบ (System Tray — เอาเมาส์
   ชี้ดูปี/เปอร์เซ็นต์/เวลาที่เหลือได้เลยโดยไม่ต้องเปิดหน้าต่างไหน), หรือหน้าเว็บ
   ของโปรแกรมเอง — กดปุ่ม **"ย่อหน้าต่าง"** มุมขวาบนเพื่อเก็บหน้าต่างนี้ไปก่อน
   ได้ โปรแกรมยังทำงานต่อเบื้องหลัง เปิดกลับได้จากเมนู tray icon ("เปิดหน้าต่าง
   โปรแกรม") ส่วนแถบลอยถ้ารู้สึกเกะกะ กดปุ่ม 🗕 บนแถบเพื่อซ่อนได้ (เรียกคืนจากเมนู
   tray icon → "แสดงแถบความคืบหน้า") — `Ctrl+Alt+F10` = หยุด
6. **รันเสร็จแล้วจะมี notification เด้งบอกทันที** (แม้กำลังทำงานอื่นอยู่) —
   ตั้งแต่ v0.7.1 ถ้ารันจบครบไม่มีปัญหา ระบบจะ**สร้าง Result.xlsx +
   Screenshots.docx ให้เองอัตโนมัติ** ไม่ต้องกดสร้างเอง แล้วคลิกที่ tray icon →
   "เปิดโฟลเดอร์ผลลัพธ์" ได้ไฟล์ส่งงานเลย

## 6. ไฟล์ผลลัพธ์ (ปกติระบบสร้างให้เองแล้ว)

รันเสร็จครบระบบสร้าง **Result.xlsx** (ตารางสรุปทุกวันปลูก) + **Screenshots.docx**
(รวมภาพ) ให้เองอัตโนมัติ (ปิดได้ที่แท็บตั้งค่า) — ถ้าอยากสร้างซ้ำระหว่างทาง
หรือหลังแก้ไฟล์ กดปุ่มในหน้า Dashboard ได้:
- **สร้าง/อัปเดต Excel** — รวมผลทุกวันปลูกที่รันสำเร็จเป็นตาราง Result เดียว
- **สร้างไฟล์ Word (ภาพ)** — รวม screenshot ของแต่ละวันปลูกเป็นเอกสารเดียว

ทั้งสองปุ่มกดซ้ำได้ทุกเมื่อ สร้างใหม่จากข้อมูล `.txt`/screenshot ที่มีอยู่จริง
ในโฟลเดอร์ผลลัพธ์เสมอ (ไม่ต้องรันใหม่ทั้งหมดถ้าไฟล์เดิมยังอยู่)

## ปัญหาที่พบบ่อย

**เปิดโปรแกรมแล้วค้างที่ "กำลังเชื่อมต่อ..." ไม่ขยับ**
มักเกิดหลังอัปเดตเวอร์ชัน — ลองปิดหน้าต่างเบราว์เซอร์ (อย่าปิดแค่ tab ถ้าเปิดค้าง
จากเวอร์ชันเก่า) แล้วเปิดโปรแกรมใหม่อีกครั้ง ตั้งแต่ v0.5.22 เป็นต้นไปเบราว์เซอร์
จะไม่ cache หน้าเก่าค้างแบบนี้แล้ว

**กด "เริ่มประมวลผล" แล้วขึ้นกล่องเตือนว่าบางวันปลูก "ไม่มีไฟล์"**
แปลว่าสถานี climate/rain ที่เลือกไว้มีข้อมูลเริ่มทีหลังเดือนที่ต้องการทดลองปลูก
จริง (เช่น อยากทดลองปลูกกุมภาพันธ์ แต่สถานีมีข้อมูลเริ่มเมษายนเป็นต้นไป) ระบบจะ
ไม่เดาใช้ข้อมูลเดือนอื่นแทนให้เองเด็ดขาด (จะทำให้ผลเพี้ยน) ต้องเลือกยืนยันเองว่า
จะข้ามวันปลูกเหล่านั้นไปหรือเปลี่ยนวันปลูกที่จะทดลอง

**เผลอคลิกโดน CropWat ระหว่างรัน**
โหมดเดสก์ท็อปซ่อน (ค่าเริ่มต้นตั้งแต่ v0.7.0) ตัดปัญหานี้ทิ้งทั้งก้อน — CropWat
ไม่ได้อยู่บนจอของคุณเลย คลิก/พิมพ์อะไรบนจอก็ไม่มีทางโดน (ถ้าใช้โหมดคลาสสิกอยู่
ไม่ควรใช้เครื่องระหว่างรัน)

**CropWat ค้างสนิท / crash ระหว่างรัน**
มีปุ่ม **"CropWat ค้าง? ปิดฉุกเฉิน"** ในแท็บ Dashboard (ข้างปุ่มเริ่มรัน/หยุด)
กดแล้วจะบังคับปิด CropWat ทุกตัว (รวมถึงตัวที่อยู่ในเดสก์ท็อปซ่อน) + รีเซ็ตสถานะ
โปรแกรมนี้กลับมาพร้อมเริ่มรันใหม่ได้เลย ไม่ต้องปิด-เปิดโปรแกรมนี้ทั้งตัว — ไฟล์
.txt ของวันปลูกที่ทำเสร็จไปแล้วไม่หายไปไหน กด "เริ่มประมวลผล" อีกครั้งระบบจะ
ทำต่อจากจุดที่ค้างให้เอง (หมายเหตุ: CropWat crash เองได้เป็นครั้งคราวจากบั๊ก
ภายในของมัน เช่น "Access violation" — โหมดเดสก์ท็อปซ่อนเปิด CropWat ใหม่ให้เอง
ทุกครั้งที่เริ่มรัน จึงฟื้นจากเหตุการณ์แบบนี้ได้ง่าย)

**วันปลูกล้มเหลวหมดทั้งปี ข้อความ error พูดถึงหน้าต่างโมดูลซ้ำ (TDayEToPMForm/
Duplicate window)**
เกิดจากมีหน้าต่างโมดูล Climate/Rain ค้างซ้ำ (เช่นฟอร์มเปล่า "untitled" ที่
CropWat สร้างค้างไว้ตอนเปิดไฟล์) — ตั้งแต่ v0.5.34 ระบบจะ**ปิดฟอร์มเปล่าที่ค้าง
ให้เองอัตโนมัติ** (เก็บบานที่โหลดไฟล์จริงไว้) แล้วทำงานต่อได้เลย ไม่ต้องหยุดรัน
ถ้ายังเจอ error นี้อยู่ (เจอกรณีที่ระบบเดาไม่ได้จริงๆ) ให้ปิด CropWat ให้สนิท
(ปุ่ม "CropWat ค้าง? ปิดฉุกเฉิน" ถ้ากด X ไม่ติด) แล้วเปิดใหม่เปล่าๆ ก่อนรันรอบ
ถัดไป — และรบกวนส่ง screenshot มาด้วยจะช่วยตามหาสาเหตุที่เหลือได้

**เจอ CropWat ค้าง/crash บ่อยตอนข้ามเดือน/ปี — ลองโหมดเดสก์ท็อปซ่อน (v0.6.0)**
แท็บตั้งค่า → การ์ด "โหมดเดสก์ท็อปซ่อน" → ติ๊กเปิด (ต้องตั้ง path ไฟล์ .exe ของ
CropWat ไว้ก่อนที่การ์ด "โปรแกรม CropWat 8.0") โหมดนี้จะเปิด CropWat ใน**เดสก์
ท็อปแยกที่มองไม่เห็น** — คุณใช้คอมทำงานอื่นได้เต็มที่ เผลอคลิกยังไงก็ไม่กระทบ
และระบบ**ไม่ต้องเหวี่ยงหน้าต่างหนีจอ** (ตัวจุดชนวน error ตอนสลับไฟล์) จึงเสถียร
กว่ามากเวลาข้ามเดือน/ปี — **ไม่ต้องเปิด CropWat ค้างไว้เอง** โปรแกรมเปิดให้เอง
(ทดสอบแล้วว่ารันข้ามเดือน+ข้ามปีได้โดยไม่ crash — เป็นของทดลอง ถ้าผลเพี้ยนให้
ปิดสวิตช์กลับโหมดเดิม)

**อยากรันแบบชัวร์ 100% ไม่ให้กระทบเครื่องจริงเลย (แยกขาดสุด)**
ดู `docs/vm-guide.md` — วิธีรันใน Virtual Machine (หนักกว่าโหมดเดสก์ท็อปซ่อนแต่
แยกขาดสมบูรณ์ที่สุด เผื่อกรณีโหมดเดสก์ท็อปซ่อนใช้กับเครื่อง/เวอร์ชัน CropWat
ของคุณไม่ได้)

## อัปเดตโปรแกรม

โปรแกรมเช็คเวอร์ชันใหม่ให้อัตโนมัติ ถ้ามีเวอร์ชันใหม่จะขึ้นแจ้งเตือนในตัวโปรแกรม
กดอัปเดตได้จากในนั้นเลย (ต้องต่ออินเทอร์เน็ต)
